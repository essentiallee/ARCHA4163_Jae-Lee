SPATIAL CANVASES
DIGITAL OBJECT: SPATIAL CANVASES

This assignment explores how primitive digital geometry can become spatial composition.
The 2D canvases investigate drawing and interaction through p5.js, while the 3D canvases translate those logics into volumetric space using Three.js.
The drawing evolves from the first 2D drawing into 3D, animated geometry with lighting and atmospheric effects.

2D CANVAS 01: DRAWING WITH PRIMITIVES

A static 2D drawing composed with primitive shapes, color, and spatial hierarchy.

2D CANVAS 02: INTERACTIVE OR ANIMATED DRAWING

An animated or interactive 2D drawing that changes through user's mouse cursor movement.

3D CANVAS 01: ORBITING GEOMETRY

A 3D composition using primitive geometry and an orbiting camera.

3D CANVAS 02: LIGHT, MATERIAL, AND ATMOSPHERE

A 3D spatial study using lighting, material, fog, or atmospheric effects.
