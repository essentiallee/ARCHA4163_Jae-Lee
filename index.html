Jae Hyun Lee
Digital Object 1: Spatial Canvases

This project explores how primitive digital geometry can become spatial composition. The 2D canvases investigate drawing, rhythm, and interaction through p5.js, while the 3D canvases translate those logics into volumetric space using Three.js.

2D Canvas 1: Drawing with Primitives

A static 2D drawing composed with primitive shapes, color, and spatial hierarchy.

2D Canvas 2: Interactive or Animated Drawing

An animated or interactive 2D drawing that changes through time or user input.

3D Canvas 1: Orbiting Geometry

A 3D composition using primitive geometry and an orbiting camera.

3D Canvas 2: Light, Material, and Atmosphere

A 3D spatial study using lighting, material, fog, or atmospheric effects.
